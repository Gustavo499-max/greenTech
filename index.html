Dados Seguros na Nuvem

Toda leitura dos sensores é enviada e armazenada em tempo real com histórico completo.

🔥
Firebase Realtime DB
Banco de dados NoSQL em nuvem do Google. Sincronização instantânea, offline-first e latência ultra-baixa para dados IoT.
JSON Realtime Google Cloud
📡
Protocolo MQTT
Lightweight messaging ideal para dispositivos IoT. Publish/Subscribe com broker Mosquitto ou HiveMQ, garantindo entrega confiável dos dados.
MQTT 3.1 Broker QoS
📊
Grafana / Dashboard
Visualização profissional com gráficos históricos, alertas automáticos por e-mail/WhatsApp quando valores saem do ideal.
